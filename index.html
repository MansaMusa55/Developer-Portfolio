Taco Cat Coding Challenge

Challenge Description

Welcome to Tacocat, the application that reverses a string and spits it back to you. Along with your reversed string the app will also determine if the string was a palindrome or not. A palindrome is defined as a word, phrase, or set of numbers that read the same backward as it does forward. Javascript was used to create a function suitable for this task. HTML, CSS, and Bootstrap were used to build the other components of this app.
